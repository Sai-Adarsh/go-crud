### go-crud
High-performance CRUD boilerplate made with Go + MariaDB + Redis
* Run MySQL on [XAMPP](https://www.apachefriends.org/docs/) and [Redis](https://redis.io/docs/getting-started/installation/install-redis-on-windows/)
* ```go mod init go-crud```
* ```go mod tidy```
* ```go run .\main.go```
